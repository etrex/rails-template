# 簡介

此 rails template 設計可快速滿足以下需求：

- 官方網站
- 基本的管理後台
- 有前後台的登入系統
- 串接 LINE 聊天機器人

使用的指令：

```
-d postgresql
-m https://raw.githubusercontent.com/etrex/rails-template/main/template.rb
```


```
rails new blog \
  -d postgresql \
  -m https://raw.githubusercontent.com/etrex/rails-template/main/template.rb
```
